開發Arduino 有很多的軟體我就介紹我最常使用的二個IDE

## 1. Arduino官方的IDE

- 這個IDE是由Arduino官方社群維護，所以比較好入門
- 官網-[Ardino](https://www.arduino.cc/)
- 下載頁面-[Downloads](https://www.arduino.cc/en/software)

![Ardiono官方網站截圖](/arduino/Arduinohomepage.png)

![Ardiono官方網站截圖](/arduino/Arduinodownloadpage.png)

## 2. VSCode + PlatformIO
- 首先電腦要先安裝Microsoft Visual Studio Code
- Microsoft Visual Studio Code(下載頁面:[VS Code](https://code.visualstudio.com/))
- 進入VSCode點選左邊直列的延伸模組搜尋PlatformIO並安裝
